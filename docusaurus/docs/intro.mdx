## Give your AI agents codebase memory

Clean indexes your repositories and lets AI coding tools search your codebase **by meaning, not just keywords**. It understands functions, classes, call graphs, and relationships between code — so your agents find the right context on the first try.

<Card title="Get running in 5 minutes" href="/getting-started/quickstart">

## Why Clean

<Columns cols={2}>

## How it works

```
 Your repo          Clean                       Your AI agent
 ─────────     ─────────────────              ────────────────
              1. SCAN files
  .py .ts ──► 2. PARSE with tree-sitter
  .js .tsx     3. EMBED with all-MiniLM-L6-v2
              4. STORE in LanceDB
              5. COMPUTE call graph
                        │
                        ▼
              Semantic vector index  ◄──  "find auth logic"  ◄──  Claude Code
                        │                                         Cursor
                        ▼                                         Windsurf
              Ranked results + context ──────────────────────►   Any MCP client
```

Clean connects to AI agents via the **Model Context Protocol (MCP)** — the open standard for giving AI tools access to external data. Agents call `search_code` and get back ranked, context-expanded results in token-optimized format.
