---

## Endpoint

```
GET /repos/{owner}/{repo}/status
```

## Authentication

Requires Bearer token with `search` scope.

## Path parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `owner` | string | Repository owner |
| `repo` | string | Repository name |

## Example request

```bash
curl "http://localhost:8000/repos/facebook/react/status" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Response

### Ready

```json
{
  "repo": "facebook/react",
  "status": "ready",
  "entity_count": 1247,
  "last_indexed_at": "2026-01-15T10:30:00Z"
}
```

### Indexing in progress

```json
{
  "repo": "facebook/react",
  "status": "indexing",
  "entity_count": 0,
  "last_indexed_at": null,
  "job_status": "running",
  "job": {
    "current_phase": "embedding",
    "phase_progress": 72,
    "files_processed": 845,
    "files_total": 1173,
    "entities_found": 2341,
    "started_at": "2026-01-15T10:35:00Z",
    "duration_seconds": 45.2,
    "api_key_id": null
  }
}
```

### Error

```json
{
  "repo": "facebook/react",
  "status": "error",
  "entity_count": 0,
  "last_indexed_at": null,
  "error": "Indexing timed out"
}
```

### Status values

| Status | Meaning |
|--------|---------|
| `cloning` | Repository is being cloned from GitHub |
| `indexing` | Parsing, embedding, and storing code entities |
| `ready` | Index is complete and searchable |
| `error` | Indexing failed (see `error` field) |

### Job phases

| Phase | Description |
|-------|-------------|
| `scanning` | Walking the file tree |
| `parsing` | Extracting code entities with tree-sitter |
| `embedding` | Generating vector embeddings |
| `storing` | Upserting into LanceDB |
| `call_graph` | Computing caller/callee relationships |

## Related endpoints

| Method | Path | Description |
|--------|------|-------------|
| `POST` | `/repos/{owner}/{repo}/cancel` | Cancel an in-progress job |
| `DELETE` | `/repos/{owner}/{repo}` | Delete the repo from the index |
