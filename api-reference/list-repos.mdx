---
title: "List repositories"
description: "GET /repos — list all indexed repositories."
---

## Endpoint

```
GET /repos
```

## Authentication

Requires Bearer token with `search` scope.

## Parameters

None.

## Example request

```bash
curl "http://localhost:8000/repos" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Response

```json
{
  "repos": [
    {
      "repo": "owner/repo",
      "branch": null,
      "status": "ready",
      "entity_count": 342,
      "last_indexed_at": "2026-01-15T10:30:00Z",
      "error": null,
      "description": "A web application framework",
      "primary_language": "python",
      "tags": ["python", "fastapi", "web"]
    },
    {
      "repo": "owner/frontend",
      "branch": "main",
      "status": "indexing",
      "entity_count": 0,
      "last_indexed_at": null,
      "error": null,
      "description": null,
      "primary_language": null,
      "tags": null,
      "job": {
        "current_phase": "parsing",
        "phase_progress": 45,
        "files_processed": 120,
        "files_total": 267,
        "entities_found": 89,
        "started_at": "2026-01-15T10:35:00Z",
        "duration_seconds": 12.5
      }
    }
  ]
}
```

### Response fields

| Field | Type | Description |
|-------|------|-------------|
| `repo` | string | Repository full name (`owner/repo`) |
| `branch` | string \| null | Indexed branch (null = default branch) |
| `status` | string | `ready`, `cloning`, `indexing`, or `error` |
| `entity_count` | integer | Number of indexed code entities |
| `last_indexed_at` | string \| null | ISO 8601 timestamp of last successful index |
| `error` | string \| null | Error message if status is `error` |
| `description` | string \| null | Auto-detected repository description |
| `primary_language` | string \| null | Auto-detected primary language |
| `tags` | array \| null | Auto-detected framework/technology tags |
| `job` | object \| null | Progress info for in-progress repos |
