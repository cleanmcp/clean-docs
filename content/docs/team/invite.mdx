## Invite via the dashboard

1. Open the [dashboard](https://tryclean.ai/dashboard)
2. Go to the **Team** tab
3. Click **Invite Member**
4. Copy the invite link and send it to your teammate

When they open the link, they'll be prompted to sign up (or sign in) and will automatically join your organization.

## What teammates get

- Access to the same indexed repositories
- Their own API key (generated from [Dashboard → API Keys](https://tryclean.ai/dashboard/keys))
- They copy the MCP config from the dashboard and add it to their agent, no additional setup needed

## Permissions

All team members share the same repositories and indexes. Separate API keys let you revoke access per person without affecting others.
