---
title: Get Your API Key
description: Find and copy your Clean API key from the dashboard.
---

Your API key is automatically created during the onboarding flow and displayed in the final MCP Config step. It looks like:

```
clean_sk_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

The key is embedded directly in the config snippet shown on the dashboard, you don't need to copy it separately.

## Managing keys

To create additional keys or revoke existing ones, go to [Dashboard → API Keys](https://tryclean.ai/dashboard/keys).

> **Keep it secret.** Your API key authenticates all requests to Clean. Don't commit it to version control.
