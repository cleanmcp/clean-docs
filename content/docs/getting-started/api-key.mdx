---
title: Get Your API Key
description: Find and copy your Clean API key from the dashboard.
---

After signing in, open the [dashboard](https://tryclean.ai/dashboard) and navigate to **API Keys**.

Your key looks like:

```
clean_sk_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

Copy it — you'll need it in the next step when running the installer.

> **Keep it secret.** Your API key authenticates all requests to your Clean instance. Don't commit it to version control.
