Clean is a self-hosted MCP server that gives your AI coding agent persistent, semantic memory of your codebase.

Instead of pasting files into context or hoping your agent guesses the right filename, Clean lets it search by **intent** — finding authentication logic, payment flows, or error handlers across your entire repo in milliseconds.

## How it works

1. **Index** — Clean scans your repos, parses Python/JS/TS with tree-sitter, and embeds every function and class using a local model. No API calls. Code never leaves your machine.

2. **Search** — Your agent calls `search_code` via MCP. Clean finds semantically similar code and expands results with the full call graph — callers, callees, and related functions up to 5 hops deep.

3. **Context** — Results come back in TOON format, 30–40% fewer tokens than JSON so your agent can fit more context per request.
