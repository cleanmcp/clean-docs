title: Antigravity

---

The installer writes the MCP config to `~/.gemini/settings.json` automatically.

## Manual setup

```json
{
  "mcpServers": {
    "clean": {
      "url": "http://localhost:8000/mcp/sse",
      "headers": {
        "Authorization": "Bearer YOUR_API_KEY"
      }
    }
  }
}
```

Antigravity uses SSE transport via the `url` field.

## Verify

Ask Antigravity to list its available tools. You should see `search_code`, `index_repo`, and `list_repos`.
