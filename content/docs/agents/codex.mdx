description: Connect Clean to OpenAI Codex CLI.
---

The installer writes the MCP config to `~/.codex/config.toml` automatically. Because Codex has no native SSE support, the CLI uses `npx mcp-remote` as a stdio proxy to the SSE endpoint.

## Manual setup

Add the following to `~/.codex/config.toml`:

```toml
[mcp_servers.clean]
command = "npx"
args = ["-y", "mcp-remote", "http://localhost:8000/mcp/sse", "--header", "Authorization:Bearer YOUR_API_KEY"]
```

## Verify

Run Codex and ask it to list its MCP tools. You should see `search_code`, `index_repo`, and `list_repos`.
