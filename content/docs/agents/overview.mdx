Clean connects to AI coding agents through the [Model Context Protocol (MCP)](https://modelcontextprotocol.io), an open standard for giving AI tools structured access to external services.

Once connected, your agent gains three tools:

| Tool | Description |
|------|-------------|
| `search_code` | Semantic search across indexed repos |
| `index_repo` | Trigger indexing for a repository |
| `list_repos` | List all indexed repositories |

## Supported agents

| Agent | Transport | Config location |
|-------|-----------|----------------|
| [Claude Code](/agents/claude) | SSE | `~/.claude.json` |
| [Cursor](/agents/cursor) | SSE | `~/.cursor/mcp.json` |
| [Claude Desktop](/agents/claude) | stdio proxy | Claude Desktop config |
| [Antigravity](/agents/antigravity) | SSE | `~/.gemini/antigravity/mcp_config.json` |
| [Codex](/agents/codex) | HTTP | `~/.codex/config.toml` |

## Getting your config

The dashboard generates the correct config block for each agent. Go to [tryclean.ai/dashboard/onboarding](https://tryclean.ai/dashboard/onboarding) and open the **MCP Config** step, or visit [Dashboard → API Keys](https://tryclean.ai/dashboard/keys) to get a config with a specific key.
