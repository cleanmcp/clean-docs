## `GET /search`

Search across indexed repositories by natural language query.

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `q` | string | ✓ | Natural language search query |
| `repo` | string | ✓ | Repository in `owner/repo` format |
| `k` | integer | — | Number of results (default: 10, max: 50) |
| `format` | string | — | `toon` (default) or `json` |
| `expand` | boolean | — | Include call graph context (default: true) |

### Example

```bash
curl "http://localhost:8000/search?q=authentication+middleware&repo=acme/api" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### Response (JSON)

```json
{
  "results": [
    {
      "name": "authenticate_request",
      "file": "src/middleware/auth.py",
      "score": 0.92,
      "line_start": 14,
      "line_end": 38,
      "code": "def authenticate_request(request):\n  ...",
      "callers": ["handle_request", "api_router"],
      "callees": ["verify_jwt", "get_user_from_token"]
    }
  ],
  "meta": {
    "query": "authentication middleware",

  }
}
```

### Similarity scores

| Score | Meaning |
|-------|---------|
| 0.90+ | Strong match |
| 0.70–0.90 | Related |
| < 0.70 | Weak |
